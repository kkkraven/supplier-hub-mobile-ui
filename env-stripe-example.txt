# 1. Получение Stripe ключей:
#    - Зайдите в https://dashboard.stripe.com/
#    - В разделе "Developers" → "API keys"
#    - Скопируйте Publishable key и Secret key
#
# 2. Создание Stripe Price IDs:
#    - В разделе "Products" создайте продукты для каждого плана
#    - Для каждого продукта создайте цены (monthly и annual)
#    - Скопируйте Price IDs в соответствующие переменные
#
# 3. Настройка Webhook:
#    - В разделе "Developers" → "Webhooks"
#    - Создайте endpoint: https://yourdomain.com/api/webhooks/stripe
#    - Выберите события (см. STRIPE_INTEGRATION_SETUP.md)
#    - Скопируйте Webhook secret
